# 💰 Your Wallet

**Your Wallet** is a personal finance management application that allows users to track their income, expenses, and financial growth over time in a simple and visual way.

## 📌 Purpose

The goal of this project is to help users gain better control over their finances by tracking expenses, setting financial goals, and visualizing their financial evolution through intuitive charts.

---

## 🚀 Features

### 👤 Authentication
- User registration
- Secure login system

### 💵 Financial Management
- Input monthly income
- Track current available balance
- Continuous financial data updates

### 🏷️ Expense Management
- Create **expense tags** (e.g., food, transportation, entertainment)
- Register expenses linked to tags
- Automatic calculation of monthly expenses

### 🎯 Financial Goals
- Define a target amount (financial goal)
- Automatic calculation of:
  - Estimated time to reach the goal
  - Progress based on income and expenses

### 📊 Data Visualization
- Graph showing financial evolution since the first record
- Clear visual tracking of financial growth
- Filter data by a specific month for detailed analysis

---

## 📈 Example Workflow

1. The user creates an account and logs in
2. Inputs:
   - Monthly income
   - Current balance
3. Creates expense categories (tags)
4. Registers expenses over time
5. Sets a financial goal
6. Tracks:
   - Estimated time to reach the goal
   - Financial growth through charts

---

## 🧠 Business Rules

- Expenses are accumulated monthly
- Goal estimation is calculated based on:
  - Income
  - Average expenses
- Charts consider the user's full financial history
- Data can be filtered by month
